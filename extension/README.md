# CPTracker Browser Extension

A browser extension for automatically extracting competitive programming problems from Codeforces and AtCoder, and importing them to CPTracker.

## Features

- **Auto-extract problem info**: Extract problem name, rating, tags, and solved time from Codeforces/AtCoder problem pages
- **Auto-fetch difficulty**: Get problem rating automatically via clist.by API
- **Code import**: Import your solution code along with the problem
- **One-click sync**: Save problems directly to your CPTracker account

## Supported Platforms

- Codeforces
- AtCoder

## Usage

1. Install the extension in your browser
2. Configure your clist.by credentials in settings (optional, for auto-fetching difficulty)
3. Visit a problem page on Codeforces or AtCoder
4. Click the extension icon to import the problem
